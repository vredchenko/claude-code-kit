# claude-tools

List of CLI tools for Claude to use, such as gron, mc, eza, etc - which can be more efficient than the default set that comes with Claude out of the box.

## Installation

```bash
/plugin marketplace add vredchenko/claude-code-kit
/plugin install claude-tools@claude-code-kit
```

## Commands

### `/claude-tools:setup`
Set up Claude Code configuration for the current project.

### `/claude-tools:context`
Generate or update the CLAUDE.md context file for the project.
