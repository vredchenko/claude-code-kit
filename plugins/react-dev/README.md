# react-dev

Plugin to guide React web development, preferred dependencies and conventions.

## Installation

```bash
/plugin marketplace add vredchenko/claude-code-kit
/plugin install react-dev@claude-code-kit
```

## Commands

### `/react-dev:component`
Generate a new React component.

### `/react-dev:hook`
Generate a new React hook.

### `/react-dev:test`
Run React component tests.
