# cli-dev

Guidelines for CLI development. Conventions, npm and Python libs, and use zx npm instead of bash where possible.

## Installation

```bash
/plugin marketplace add vredchenko/claude-code-kit
/plugin install cli-dev@claude-code-kit
```

## Commands

### `/cli-dev:init`
Initialize a new CLI application project.

### `/cli-dev:command`
Add a new command to the CLI application.
