# python-dev

Plugin to guide Python development - Python version, code style, uv and ecosystem, third-party libraries of choice.

## Installation

```bash
/plugin marketplace add vredchenko/claude-code-kit
/plugin install python-dev@claude-code-kit
```

## Commands

### `/python-dev:init`
Initialize a new Python project with modern tooling.

### `/python-dev:test`
Run Python tests and analyze results.

### `/python-dev:lint`
Run linting and formatting checks on Python code.
