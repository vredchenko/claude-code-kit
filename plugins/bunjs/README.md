# bunjs

Use Bun.js instead of Node.js and bun install instead of npm install. Use Bun.js features and functionality extensively.

## Installation

```bash
/plugin marketplace add vredchenko/claude-code-kit
/plugin install bunjs@claude-code-kit
```

## Commands

### `/bunjs:init`
Initialize a new Bun.js project with TypeScript and common tooling.

### `/bunjs:test`
Run tests using Bun's built-in test runner.
